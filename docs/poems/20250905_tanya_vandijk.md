# Ходи пеша, подари вдъхновение

!!! note "Таня ван Дайк (5 Септември 2025)"

Ходи пеша, подари вдъхновение  
Остави колата — върви с нетърпение,  
по улици тихи в големия град.  
Децата ни чакат — с усмивки, с видения  
да дишат живота без дим и без страх.

В крачките твои се крие промяната,  
във всяка стъпка — надежда блести.  
Когато вървим, светът се събужда,  
а въздухът чист за тях се роди.

Не е нужно много — само желание,  
да бъдеш пример, да водиш с любов.  
Ходи пеша, бъди вдъхновение  
за бъдеще светло, за детски живот.
